Prerequisities:
    Python 3.5
    pywin32: https://sourceforge.net/projects/pywin32/files/pywin32/

    pip install pyserial pyfrc numpy crccheck pynetworktables
    pip install --upgrade  --trusted-host wxpython.org --pre -f http://wxpython.org/Phoenix/snapshot-builds/ wxPython_Phoenix

Checkout this repository. Navigate to the Python folder and run:
    python setup.py install

How to start the program:
    After the install, there will hopefully be a link on your desktop. 
    C:\Python35\Scripts\ControlBoardApp.exe
