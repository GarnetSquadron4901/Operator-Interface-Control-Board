Prerequisities:

pywin32: https://sourceforge.net/projects/pywin32/files/pywin32/

pip install pyserial, pyfrc, numpy
pip install -U --pre pynetworktables
pip install --upgrade  --trusted-host wxpython.org --pre -f http://wxpython.org/Phoenix/snapshot-builds/ wxPython_Phoenix

How to install:
python setup.py install

How to start the simulator:
C:\Python35\Scripts\ControlBoardApp.exe simulator
